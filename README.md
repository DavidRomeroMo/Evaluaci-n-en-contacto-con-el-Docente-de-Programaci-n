Aquí estarán todos los documentos necesarios para completar la evaluación en contacto con el Docente

Nombre del Proyecto: Control de inventario para una tienda.
Integrantes: David Agustin Romero Moncayo.
Objetivo del sistema: esarrollar un sistema de control de inventario para una tienda que permita registrar productos, administrar el stock, controlar las ventas y supervisar las fechas de caducidad, facilitando una gestión más organizada, eficiente y segura del inventario.
Descripcion de Funcionalidades: Registrar nuevos productos con nombre, precio, cantidad y fecha de caducidad.
Visualizar el inventario completo de forma organizada.
Registrar ventas y actualizar automáticamente el stock disponible.
Modificar la cantidad de productos en inventario.
Identificar productos agotados para facilitar su reposición.
Detectar productos próximos a caducar para evitar pérdidas.
Buscar productos rápidamente por su nombre.
Validar los datos ingresados para reducir errores en el registro.
